Sistema de Ventas JavaFX

## Descripción

Aplicación desarrollada en JavaFX para la gestión básica de un sistema de ventas. Permite el acceso mediante inicio de sesión y muestra diferentes interfaces según el rol seleccionado.

## Tecnologías Utilizadas

* Java
* JavaFX
* Maven

## Funcionalidades Implementadas

* Inicio de sesión.
* Selección de rol (Administrador, Vendedor y Cajero).
* Navegación entre pantallas según el rol.
* Interfaz gráfica desarrollada con JavaFX.

## Ejecución del Proyecto

1. Clonar el repositorio.
2. Abrir el proyecto en IntelliJ IDEA.
3. Descargar las dependencias Maven.
4. Ejecutar la clase `HelloApplication`.
5. Iniciar sesión con las credenciales configuradas en el sistema.
